# Capacitor icon and splash placeholders

Place your branding assets here before generating final Android icons/splash resources.

- `icon.png`: 1024x1024 (app icon source)
- `splash.png`: 2732x2732 (splash screen source)
- `adaptive-icon.png`: 1024x1024 (Android adaptive foreground)
- `adaptive-icon-background.png`: 1024x1024 (Android adaptive background)

You can then generate native resources with:

```bash
npx @capacitor/assets generate --android
```
